ИНСТРУКЦИЯ ДЛЯ ЗАПУСКА AI ПАРТНЁРСКОГО АССИСТЕНТА

1. Убедись, что установлен Node.js:
   - Скачай с https://nodejs.org  (LTS версия)
   - Установи с галочкой "Add to PATH"

2. Открой папку проекта в VS Code

3. В меню: Terminal → New Terminal

4. Выполни команды:
   npm install
   npm start

5. Браузер откроет страницу: http://localhost:3000

Дополнительно:
- Все данные шифруются перед сохранением
- ИИ анализирует совместимость офферов
- Поддерживается интеграция с Supabase
- Можно добавить автономную регистрацию и публикации
